# Contributing Guidelines

Thank you for considering contributing to this project!

## How to Contribute

1. **Fork the Repository**  
   Create a personal fork of the repository on GitHub.

2. **Create a Branch**  
   Make your changes in a new branch based on the latest `master` branch.

3. **Submit a Pull Request (PR)**  
   Open a pull request from your branch to the main repository.  
   Please provide a clear description of your changes.

   Contributions that alter language semantics or compiler architecture may require additional discussion before acceptance.

4. **Review Process**  
   Your PR will be reviewed. Please address any feedback and update your PR as needed.

## License Compatibility

By submitting a contribution, you agree that your code will be licensed under the MIT license, consistent with the project. You represent that you have the right to submit the contribution and that it does not violate any third-party rights.
